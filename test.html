Antenna & Radiation Formula Reference

Compact Formula & Concept Table

Fundamental Concepts & Power Relations

Concept Formula(s) Notes / Units
Instantaneous Poynting Vector $$\vec{S}(t) = \vec{E}(t) \times \vec{H}(t)$$ W/m², instantaneous power flux density vector
Time-Average Power Density $$W_{av} = \frac{1}{2}\text{Re}\{\vec{E} \times \vec{H}^*\}$$ W/m² (using complex phasors)
For Plane Wave $$W = \frac{|E|^2}{2\eta_0}$$ where $\eta_0 = 377\,\Omega$ Free space wave impedance
Surface Element (Sphere) $$d\vec{S} = r^2 \sin\theta \, d\theta \, d\phi \, \hat{a}_r$$ m² (vector points radially outward)
Solid Angle Element $$d\Omega = \sin\theta \, d\theta \, d\phi$$ steradian (sr)

Radiation Patterns & Directivity

Concept Formula(s) Notes / Units
Radiation Intensity $$U(\theta,\phi) = r^2 W(\theta,\phi)$$ W/sr — power per unit solid angle
Total Radiated Power $$P_{rad} = \iint_{4\pi} W \, dS = \iint_{4\pi} U \, d\Omega$$ W; integrate $\theta \in [0,\pi]$, $\phi \in [0,2\pi]$
Normalized Power Pattern $$P_n(\theta,\phi) = \frac{U(\theta,\phi)}{U_{max}}$$ Dimensionless; $\Omega_A = \int_{4\pi} P_n \, d\Omega$
Beam Solid Angle $$\Omega_A = \int_{4\pi} \frac{U}{U_{max}} \, d\Omega = \frac{P_{rad}}{U_{max}}$$ steradian (sr)
Directivity (Linear) $$D_0 = \frac{4\pi U_{max}}{P_{rad}} = \frac{4\pi}{\Omega_A}$$ unitless
Directivity (dB) $$D_0(\text{dB}) = 10\log_{10} D_0$$ referenced to isotropic (dBi)

Beamwidth Definitions

Concept Formula(s) Notes / Units
Half-Power (-3 dB) Condition $$U(\theta_H) = \frac{U_{max}}{2}$$ Solve for $\theta_H$ in chosen plane
Half-Power Beamwidth (HPBW) $$\text{HPBW} = \theta_2 - \theta_1$$ Angular separation between half-power points
First Null Beamwidth (FNBW) Angular separation between first nulls ($U=0$) degrees, in main-lobe plane
Approximate Directivity from Beamwidths $$D_0 \approx \frac{4\pi}{\theta_{HP} \phi_{HP}}$$ (radians)

$$D_0 \approx \frac{41253}{\text{HPBW}_{az}(\deg) \cdot \text{HPBW}_{el}(\deg)}$$		 Useful approximation for main lobe

Antenna Parameters & Efficiency

Concept Formula(s) Notes / Units
Gain $$G = \eta D_0$$ or $$G_{\text{dB}} = 10\log_{10} G$$ $\eta$ = radiation efficiency (0–1)
Antenna Efficiency $$\eta = \frac{P_{rad}}{P_{in}}$$ fraction radiated vs input
Antenna Input Impedance $$Z_{ant} = (R_r + R_L) + jX_A$$ $R_r$ radiation resistance, $R_L$ loss resistance, $X_A$ reactance
Powers in Equivalent Circuit $$P_{rad} = \frac{1}{2}I^2R_r$$ $$P_{loss} = \frac{1}{2}I^2R_L$$ I = antenna current (RMS)

Source & Matching Relations

Concept Formula(s) Notes / Units
With Source $V_g, Z_g = R_g + jX_g$ — Current $$I_g = \frac{V_g}{(R_g + R_r + R_L) + j(X_g + X_A)}$$ Ohm's law
With Source — Radiated Power $$P_{rad} = \frac{1}{2} \frac{V_g^2 R_r}{(R_g + R_r + R_L)^2 + (X_g + X_A)^2}$$ Analogous forms for $P_{loss}$, $P_g$ by replacing numerator with $R_L$ or $R_g$
Conjugate Matching Conditions $$X_g = -X_A \text{ and } R_g = R_r + R_L$$ For maximum power transfer
Fraction of Generator Power Radiated $$\frac{P_{rad}}{P_{total}} = \frac{R_r}{R_g + R_r + R_L}$$ Useful when comparing losses & design choices

Transmission & Reception

Concept Formula(s) Notes / Units
Aperture / Effective Area $$A_e = \frac{\lambda^2}{4\pi} G$$ m² — how much power a receiving antenna captures from plane wave
Friis Transmission (Far Field) $$P_r = P_t G_t G_r \left(\frac{\lambda}{4\pi R}\right)^2$$ Free-space power transfer (use matched polarization)

Common Conversions & Plotting Conventions

Concept Formula(s) Notes / Units
Typical Conversion 1 sr = steradian
$4\pi$ sr = full sphere		 Keep units consistent (rad vs deg)
Usual Plotting Planes E-plane: contains $\vec{E}$ and boresight
H-plane: contains $\vec{H}$ and boresight		 Measure HPBW & pattern cuts in these planes

Quick Solved Examples (Useful for Exams)

Example 1: $U(\theta) = A_0 \sin\theta$

$$P_{rad} = \frac{\pi}{2}A_0, \quad U_{max} = A_0, \quad D_0 = \frac{4\pi A_0}{\frac{\pi}{2}A_0} = \frac{4}{\pi} \approx 1.273 \text{ (= 1.05 dBi)}$$

HPBW: Half-power when $\sin\theta = 1/2 \Rightarrow \theta = 30°, 150° \Rightarrow$ HPBW = 120°

FNBW: First nulls at 0° and 180° → FNBW = 180°

Example 2: $U(\theta) = A_0 \sin^2\theta$

$$P_{rad} = \frac{8\pi}{3}A_0, \quad D_0 = \frac{4\pi A_0}{(8\pi/3)A_0} = \frac{3}{2} = 1.5 \text{ (1.76 dBi)}$$

Example 3: $U(\theta) = A_0 \sin^3\theta$

$$P_{rad} = \frac{3\pi^2}{4}A_0, \quad D_0 = \frac{4\pi}{3\pi^2/4} = \frac{16}{3\pi} \approx 1.698 \text{ (2.30 dBi)}$$

Memory Tips (One-Line Each)

  • Poynting → W (avg) → multiply by $r^2$ → U
  • Integrate U over $4\pi$ → $P_{rad}$
  • $D_0 = 4\pi U_{max}/P_{rad} = 4\pi/\Omega_A$
  • Gain = η·Directivity ; $A_e = \lambda^2 G/(4\pi)$
  • HPBW: find U = $U_{max}/2$ in that plane; FNBW: first U = 0
